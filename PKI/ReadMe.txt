当前路径MODPKICA/PKI
--------------------------------
本文件解释当前路径目录下的文件作用和结构
****************************************************************
类型      名称      描述 
----------------------------------------------------------------
文件夹     ROOT       该文件夹主要储存根证书crt文件、根私钥key文件
				     内部文件名格式 root.crt root.key
-----------------------------------------------------------------
文件夹     CA         该文件夹主要储存中间CA证书的crt文件、私钥key文件
				     内部文件名格式 ca.crt ca.key

文件夹     OCSP       该文件夹主要储存OCSP专用签名证书的crt文件、私钥key文件
				     内部文件名格式 ocsp.crt ocsp.key
-------------------------------------------------------------------
文件夹     CERT       该文件夹主要储存 用户证书(最终实体)的crt文件
				     内部文件名格式 证书序列号.crt
------------------------------------------------------------------
文件夹     KEY        该文件夹主要储存 用户证书(最终实体)的私钥key文件
				     内部文件名格式 证书序列号.key
------------------------------------------------------------------
文件夹     TIMSTAMP   该文件夹主要储存时间戳专用签名证书的crt文件、私钥key文件
				     内部文件名格式 tsasha1.crt  tsasha1.key
								 tsasha256.crt tsasha256.key
-------------------------------------------------------------------
文件夹     WebPublic  该文件夹主要供PKI基础设施和外部网络访问
					 所有crt证书副本、吊销列表crl、网页模板、时间戳服务器
					 
    子文件夹   CRT      该文件夹主要储存MODPKI系统中所有crt证书副本
					 内部文件格式 证书序列号.crt root.crt ca.crt *.crt
   ----------------------------------------------------------------
    子文件夹   CRL      该文件夹主要储存MODPKI系统中所有证书吊销列表
					 内部文件格式 root.crl  ca.crl  *.crl
   ----------------------------------------------------------------
    子文件夹   WWW      该文件夹主要储存MODPKI系统网页源码
-------------------------------------------------------------------
文件     auto.exe    该文件主要作用处理业务：吊销证书、增加证书、
					自动化管理内部证书库(CERTS.txt) 
-------------------------------------------------------------------
文件     auto.go     该文件是go语言源码编译出 auto.exe	
-------------------------------------------------------------------

文件  CERTS.txt	该文件主要记录MODPKI系统中所有证书信息
		    格式 证书序列号 证书类别  证书状态   吊销原因编号  签发/吊销时间
				证书序列号 十六进制数值文本
				证书类别 R:根证书 C:中间CA证书 E:用户证书 
				证书状态 V:正常 R:已吊销  N:未知/存疑  可自行扩展
				吊销原因编号 0~9 数字整数
				第五项记录证书 签发时间 或 吊销时间 （目前二选一，可自行扩展）
-------------------------------------------------------------------
文件  CERTS.txt.sm    该文件是上述CERTS.txt的格式模板文件
					  1.执行操作初始化后将会刷新现有CERTS.txt
					  2.自动会备份现有数据到CERTS.txt.old
-------------------------------------------------------------------
文件   CONFIG         该文件是MODPKI系统的配置文件
					 管理颁发证书的配置模板
					 管理自动化程序和网页端口
-------------------------------------------------------------------
文件   ReadMe.txt     你正在查看........
