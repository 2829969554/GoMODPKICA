# MODPKICA

#### 介绍
MOD PKI CA是一个Go语言编写的开放源码项目，目的构建一个功能齐全的证书系统。是一种自由和开源 PKI (公钥基础建设) 软RA Server、CA Server、 OCSP Server、Timestamp Server、CT logs Server，目的是打造全自动化的国产证书系统，目前支持RSA算法证书，即将支持ECC、国密SM算法、DSA算法、目前是一边学习一边开发，进度会比较慢
